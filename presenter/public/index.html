Remote Self-Host Query

If the user or usher UI is hosted on another device, include ?ws=ws://PRESENTER_IP:5050 so it connects back to the presenter server.

Quick Start

  1. Open Presenter tab and keep it visible.
  2. Open one or more User tabs with different IDs/seats.
  3. Open Usher tab from presenter host link (no Playwright needed).
  4. For AWS blinking automation, run user script: npm run start:aws-guide.
  5. Advance and resolve checkpoints from the role UIs.

# Presenter Modules

This folder contains documentation for reusable modules (libraries) used by the presenter-side application. Each markdown file here mirrors a code module, but tells the story of what the module does and how it supports the presenter's role.

## How to Read
- Each file describes a module's purpose, how it fits into the presenter's workflow, and its main responsibilities.
- The documentation is written as a story, making it easy to understand the role of each module before diving into code.

---

*Modules are the tools that empower the presenter to guide, monitor, and support the workshop.*

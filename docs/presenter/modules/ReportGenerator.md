# ReportGenerator Module

The ReportGenerator module is the presenter's storyteller. It gathers all the data from the workshop—who needed help, who finished first, and how the group did overall—and turns it into a clear, useful report.

## Story
At the end of the workshop, ReportGenerator collects every moment: each step completed, every help request, and all resolutions. It weaves these into a summary that helps the presenter reflect and improve for next time.

## Main Responsibilities
- Collect progress and help data
- Summarize workshop outcomes
- Export reports for review

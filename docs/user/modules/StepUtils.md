# StepUtils Module

The StepUtils module is the user's stepwise guidebook. It helps the app know which step comes next, which elements to highlight, and how to check if the user has completed all required actions.

## Story
Whenever the user moves through the workshop, StepUtils quietly checks off each requirement. It knows the order of steps and what needs to be done, making sure the user never skips ahead or gets lost.

## Main Responsibilities
- Track the current and next steps
- List required UI actions for each step
- Confirm when all actions are complete

---

*StepUtils is the user's backstage manager, making sure every cue is hit before the show goes on.*

# HelpRequest Module

The HelpRequest module is the user's messenger. It handles sending and tracking help requests, ensuring the user always has a way to reach out for assistance.

## Story
When the user clicks the help button, HelpRequest springs into action, sending a message to the usher and waiting for a response. It keeps the user informed, so they know help is on the way.

## Main Responsibilities
- Send help requests to the usher
- Track outstanding requests
- Notify the user when help is coming

---

*HelpRequest is the user's walkie-talkie, always ready to call for backup.*

# ResolutionNotifier Module

The ResolutionNotifier module is the usher's confirmation system. It lets the presenter know when a participant's concern has been resolved, closing the loop on every help request.

## Story
After helping a participant, the usher uses ResolutionNotifier to send a quick message to the presenter. This way, the presenter's records are always up to date, and the workshop's story is complete.

## Main Responsibilities
- Notify presenter when a concern is resolved
- Ensure all resolutions are recorded
- Keep the help process transparent

---

*ResolutionNotifier is the usher's handshake, confirming every participant gets the help they need.*

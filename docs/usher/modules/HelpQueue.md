# HelpQueue Module

The HelpQueue module is the usher's to-do list. It keeps track of every help request, making sure no participant is left waiting.

## Story
As help requests come in, HelpQueue lines them up, showing the usher who needs help and in what order. When a concern is resolved, it moves on to the next, keeping the workshop running smoothly.

## Main Responsibilities
- Track all outstanding help requests
- Prioritize and order requests
- Remove resolved concerns

---

*HelpQueue is the usher's queue manager, making sure every call for help is answered in turn.*

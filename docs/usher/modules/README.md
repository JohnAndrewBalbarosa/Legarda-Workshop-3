# Usher Modules

This folder contains documentation for reusable modules (libraries) used by the usher-side application. Each markdown file here mirrors a code module, but tells the story of what the module does and how it helps the usher assist participants.

## How to Read
- Each file describes a module's purpose, how it fits into the usher's workflow, and its main responsibilities.
- The documentation is written as a story, making it easy to understand the role of each module before diving into code.

---

*Modules are the usher's toolkit, making sure every participant gets timely and effective help.*
